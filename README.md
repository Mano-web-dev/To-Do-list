# To-Do List

A modern, responsive to-do list application built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Delete tasks
- Fully responsive design
- Modern UI with gradients and animations
- Clean and intuitive interface

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6)

## How to Use

1. Open https://mano-web-dev.github.io/To-Do-list/
2. Type a task in the input field
3. Click "Add Task" or press Enter to add the task
4. Click the "Ã—" button to delete a task
